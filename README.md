# Kanbas Node.js Back-end
Back-end of Kanbas, a Canvas like web app for students to manage their assignments, courses and quizzes.  
Students can enroll in new classes, drop out of classes, view their modules and assignments, and take multipage quizzes with different types of questions.  
Faculty can add quizzes and new course content to classes and manage enrolled students.  
Admins have further control such as deleting classes. 

Uses the Kanbas React Front-end: https://github.com/cuanfraser/kanbas-react-web-app
